# OpenStream Landing

Landing page premium para OpenStream y Senal del Pueblo, construida con React, Vite, Tailwind CSS, Framer Motion y Lucide React.

## Comandos

```bash
npm install
npm run dev
npm run build
```

Servidor local:

```bash
http://127.0.0.1:5173
```

## Enlace de la Web App

Por defecto el CTA apunta a `#web-app`. Para usar tu reproductor real, crea un archivo `.env`:

```bash
VITE_WEB_APP_URL=https://tu-dominio-del-reproductor.com
```

## Instaladores

Coloca los archivos reales en:

```bash
public/downloads/OpenStream-Mobile.apk
public/downloads/OpenStream-Desktop.exe
```

La landing ya tiene los botones conectados a esas rutas.
